LinkGuard

LinkGuard tracks file changes to help recover moved, renamed or deleted assets in your creative projects. Protect your videos from broken links and lost files.

LinkGuardは、ファイルの移動・名前変更・削除を追跡し、映像制作プロジェクトのリンク切れやファイル紛失から守るツールです。

Download Source (ZIP) Download for Windows Download for macOS

Features / 機能

  • Monitors folders and logs file operations (delete, move, rename, update)
  • フォルダを監視し、削除・移動・名前変更・更新を記録します
  • Tracks files by their fingerprints to locate them even after they move
  • ファイル指紋で追跡し、移動後でも見つけられます
  • Recreates old paths using symlinks/junctions to restore project links
  • シンボリックリンクやジャンクションを使って旧パスを再現し、リンクを復元します
